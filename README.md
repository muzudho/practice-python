# git 操作

```shell
# リモート・レポジトリーの更新を受信する
git pull

# カレント・ディレクトリーにある、ローカルＰＣのレポジトリに追加されているすべてのファイルをコミット
git commit -a

# コミット用のコメント記入
# [Ctrl] + [O] で COMMIT_EDITMSG というファイル名で保存
# [Ctrl] + [X] でエディター終了

git push
```
